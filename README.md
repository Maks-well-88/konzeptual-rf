# Новостной портал «Концептуал.рф»
[«Концептуал.рф»](https://xn--80ajoghfjyj0a.xn--p1ai/) &mdash; СМИ патриотического издательства «Концептуал». Здесь можно познакомиться с экспертным мнением ведущих российских историков, политологов и экономистов по актуальным проблемам современности.

## Набор работ
В течение недели я самостоятельно разработал новый дизайн для портала с учетом требования адаптивности (монитор, ноутбук, планшет, смартфон). Изначально модель проекта была написана на HTML, CSS и ванильном JS. Для продакшена на WordPress переделан на jQuery.
